# FINTECH IT SOLUTIONS Full Stack Project

## Backend
- Node.js + Express
- SQLite database preloaded (fintech.db)
- Run: 
    cd backend
    npm install
    node server.js

## Frontend
- React + Tailwind CSS
- Run:
    cd frontend
    npm install
    npm start

## Features
- Employee, Project, Payroll management
- Contact form
- Admin & Employee dashboards
- JWT Authentication
